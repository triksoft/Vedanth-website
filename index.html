PROJECT: MED_BOX_DISPENSER

1. PROJECT ABSTRACT

The Automated Medicine Dispenser is an IoT-based system designed to assist elderly patients with medication adherence. It automates the pill dispensing process at scheduled intervals to prevent dosage misses.

2. HARDWARE SPECIFICATIONS
  • > Microcontroller: Arduino Uno / Nano
  • > RTC Module: DS3231 (Real Time Clock)
  • > Actuator: Servo Motor (MG996R)
  • > Display: LCD 16x2 I2C
  • > Alert System: Piezo Buzzer & LED Indicators
3. WORKING PRINCIPLE

The system initializes by reading the current time from the RTC module. The microcontroller compares this real-time data with the pre-defined medication schedule arrays.

When a match occurs (Alarm Condition), the Servo Motor rotates to a calculated angle (e.g., 45 degrees) to open the specific pill compartment. Simultaneously, the buzzer sounds an alert, and the LCD displays the medicine name.

4. CONCLUSION & FUTURE SCOPE

This prototype successfully demonstrates a low-cost, effective solution for healthcare automation. Future iterations could include GSM modules for SMS alerts to caregivers or cloud integration for remote monitoring.
